# javascript 연산자


아래는 자바스크립트에서 사용되는 일부 연산자의 우선순위와 결합성입니다.

* 그룹화 연산자: ()
* 멤버십 연산자: . []
* 증감 연산자: ++ --
* 단항 연산자: + - ! ~ typeof void delete new
* 산술 연산자: * / % + -
* 시프트 연산자: << >> >>>
* 관계 연산자: < <= > >= instanceof in
* 등호 연산자: == != === !==
* 논리 연산자: && ||
* 조건 연산자: ? :
* 할당 연산자: = += -= *= /= %= <<= >>= >>>= &= ^= |=

우선순위가 높은 연산자가 먼저 계산되며, 우선순위가 같은 연산자는 결합성에 따라 왼쪽에서 오른쪽 또는 오른쪽에서 왼쪽으로 계산됩니다.
